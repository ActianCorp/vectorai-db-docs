## What is VectorAI DB?

Traditional databases focus on exact-match queries for structured data. AI applications require similarity search on dense vector representations of unstructured data such as text, images, and audio.

### Purpose-built for AI

Native vector operations from ingestion to retrieval. No extensions or workarounds needed.

### AI beyond the cloud

Deploy where your data lives: edge devices, embedded systems, or on-premises infrastructure. Full AI capabilities without cloud dependency.

### Developer-first API

Intuitive Python SDK with async or sync clients, type-safe filter DSL, and smart batching for high productivity.

### High performance

HNSW indexing powered by FAISS delivers sub-millisecond approximate nearest neighbor search across millions of vectors.
