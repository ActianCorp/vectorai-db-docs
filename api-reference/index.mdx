## REST API

The REST API provides a familiar HTTP interface with JSON payloads. It runs on port 50052 by default and supports all core operations including collection management, vector search, and data manipulation. REST is the easiest way to get started and works with any HTTP client or programming language.

## gRPC

gRPC offers superior performance with efficient binary serialization and HTTP/2 multiplexing. It runs on port 50051 by default and is the recommended choice when you need minimal latency, bidirectional streaming, or are building performance-critical applications. The Python SDK uses gRPC by default.
